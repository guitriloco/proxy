Automação via Proxy Auto-Config (Arquivo PAC)Se você quiser que o Firefox use o proxy apenas para sites específicos (ex: só para o TikTok ou só para o Habbo), você pode usar um arquivo .pac.Crie um arquivo chamado proxy.pac com este conteúdo:JavaScriptfunction FindProxyForURL(url, host) {
    // Se o site for um desses, use o nosso Roteador Central
    if (shExpMatch(host, "*.tiktok.com") || shExpMatch(host, "*.habbo.com.br")) {
        return "PROXY 127.0.0.1:8080";
    }
    // Para o resto, use conexão direta (sem proxy)
    return "DIRECT";
}
